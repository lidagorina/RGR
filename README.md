# RGR work — Криптографическая система

## Описание

RGR work — это консольное приложение для шифрования и дешифрования текстовых данных и файлов. Программа поддерживает шесть криптографических алгоритмов, каждый из которых реализован в виде отдельного динамически подключаемого модуля (плагина).

Пользователь может работать с программой в двух режимах:
- Интерактивный режим (меню) — для удобной работы через пошаговые подсказки.
- Режим командной строки (CLI) — для автоматизации и быстрого выполнения операций.

Программа корректно обрабатывает русские буквы благодаря функции `splitUtf8`, которая разбивает строку на символы, а не на байты. Это позволяет шифровать текст на русском языке без искажений.

Основные возможности:
- Шифрование и дешифрование текста, введённого с клавиатуры.
- Шифрование и дешифрование файлов любых типов
- Генерация случайных ключей для выбранного алгоритма.
- Вывод зашифрованных данных в текстовом и HEX-формате.
- Гибкое управление плагинами через конфигурационный файл.
## Алгоритмы
- Цезарь — сдвиг в алфавите
- XOR — побитовое исключающее ИЛИ
- Atbash — зеркальное отображение
- ROT128 — сдвиг байтов на 128
- Playfair — шифрование по парам символов
- ChaCha20 — поточный шифр
